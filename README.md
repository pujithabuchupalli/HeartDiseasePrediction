# Heart Disease Prediction System ❤️

## 📌 Project Overview
This project predicts the presence of heart disease in patients using machine learning techniques.  
It analyzes medical parameters such as age, cholesterol, blood pressure, and ECG results to assist in early diagnosis.

## 🧠 Problem Statement
Heart disease is one of the leading causes of death worldwide.  
Early prediction can help in timely treatment and prevention.

## 📊 Dataset
- Source: UCI Heart Disease Dataset
- Attributes include:
  - Age
  - Sex
  - Chest Pain Type
  - Resting Blood Pressure
  - Cholesterol
  - Fasting Blood Sugar
  - Maximum Heart Rate
  - ECG Results
