# ❤️ Heart Disease Prediction System

## 📌 Project Overview

This project aims to predict the presence of heart disease in patients using machine learning techniques.
It analyzes key medical attributes such as age, cholesterol levels, blood pressure, and ECG results to assist in early diagnosis and risk assessment.

---

## 🧠 Problem Statement

Heart disease is one of the leading causes of death worldwide.
Early prediction and risk identification can help in timely medical intervention and improve patient outcomes.

---

## 📊 Dataset

* **Source:** UCI Heart Disease Dataset
* The dataset contains various medical attributes, including:

  * Age
  * Sex
  * Chest Pain Type
  * Resting Blood Pressure
  * Cholesterol
  * Fasting Blood Sugar
  * Maximum Heart Rate
  * ECG Results

## ✅ Model Performance

* Achieved **~85–90% accuracy**, with **Random Forest** performing the best among all models.
* Evaluated models using:

  * Accuracy
  * Precision
  * Recall
  * Confusion Matrix

---

## 🔍 Key Insights

* Features like **chest pain type** and **maximum heart rate** significantly influence prediction results.
* Ensemble models (Random Forest) provided better performance compared to individual models.
